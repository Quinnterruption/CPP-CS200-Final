# CPP-CS200-Final

## User Input
Open a `.rndr` file with the program to play back the saved recording

| Input | Descript |
| --- | --- |
| `x`/`y`/`z` | Toggles rotation about the `x`/`y`/`z`-axis |
| `←`/`→`/`↑`/`↓` | Moves object `left`/`right`/`up`/`down` |
| Scroll `Forward`/`Backward` | Moves object `forwards`/`backwards` |
